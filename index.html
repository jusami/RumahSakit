Welcome to Web Sistem Informasi Rumah Sakit

Terima Kasih kami ucapkan kepada anda yang sudah memberi waktu luang anda untuk mengunjungi website kami. Web ini sengaja kami buat untuk memenuhi tugas kami pada mata kuliah TKPPL.

Keterangan

Website Sistem Informasi Rumah Sakit bertujuan untuk membantu untuk penginputan data-data informasi tentang rumah sakit, sehingga admin rumah sakit dengan mudah untuk menginformasikan data-data pasien, dokter, suster, dll.

Kelebihan

1. Membuat database untuk seluruh pegawai rumah sakit
2. Memudahkan receptionis untuk mencari data pegawai dengan mudah

Nilai Jual

1. Web ini jelas tertera bahwa Sistem Informasi Database mudah dipahami dan sangat membantu bagi pemula
2. Sistem dari web ini tidak membingungkan bagi pengguna yang memakainya

Cara Penggunaan Web

1. Login Admin

2. Bagian input data, dapat menginput dokter, pasien, karyawan, dll

3. Bagian lihat merupakan informasi yang telah diinput didalam bagian input data

4. Akun, bagian untuk admin jika ingin mengganti username atau password

5. Logout, keluar dari web
